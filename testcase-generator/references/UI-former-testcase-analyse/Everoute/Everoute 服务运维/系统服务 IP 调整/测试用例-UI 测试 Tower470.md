- UI 测试 Tower470
    - ER 概览页
        - [C] 代理异常数字为 未配置 + 代理异常 总数
        - [C] 点击弹窗，分别显示 哪些主机代理异常，哪些主机未配置 TEPIP 地址
    - ER < 3.2.0
        - 虚拟管理 IP
            - [C] 未配置时，为 input 框，并存在提示文字 [配置后不可修改]
            - [C] 配置后，展示 vip 文字，无处编辑
        - ControllerIP
            - [C] 所属集群右侧无编辑按钮
        - LBVMIP
            - [C] 点击编辑 LBvm 没有 [网络设置]
        - GatewayVM
            - [C] 点击编辑网络配置，边缘网关虚拟机内部信息均不可编辑
        - 集群关联
            - [C] 点击编辑集群关联，VDS，TEPIP 池子，TEPIP 均不可编辑
            - [C] 只可编辑扩容节点的 TEPIP
        - TEPIP 池子
            - [C] 点击编辑 TEPIP 池子，子网掩码不可编辑
    - ER >= 3.2.0
        - 虚拟管理 IP（ER3.4.0 不含此项）
            - [C] 未配置时，为 input 框，不存在提示文字
            - [C] 配置后，仍为 input 框
            - [C] vip不允许被删除
        - ControllerIP
            - [C] 所属集群右侧有编辑按钮
            - [C] 若产生实质修改则显示具体更换内容二次确认弹窗
            - 仅一个 Controller
                - [C] 网关不和 VIP 相同，会报错
                - [C] 网关和 controller 不可重复，会报错
            - 三/五个 Controller
                - [C] 网关不和 VIP 相同，会报错
                - [C] 网关和 controller 不可重复，会报错
                - [C] controller 之间不可重复，会报错
                - [C] 支持批量配置 controller 新 ip
        - LBVMIP
            - ER = 3.4.0 且 LBI 开启 VPC
                - [C] TEP 虚拟机网络可编辑
                - [C] 切换 TEPIP 池清空 TEPIP，留空则自动分配
                - [C] LBI 的 lbvm 编辑虚拟机网络和 IP
                - 二次弹窗
                    - [C] 管理网络子网掩码
                    - [C] 管理网络网关
                    - [C] LB 虚拟机 1
                    - [C] LB 虚拟机 2
                    - [C] 自动分配 TEPIP
            - 3.2.0 <= ER < 3.4.0 & ER = 3.4.0 且 LBI 不开启 VPC
                - [C] 只可编辑管理虚拟机网络 vlan 和 ip
                - 二次弹窗
                    - [C] 管理网络子网掩码
                    - [C] 管理网络网关
                    - [C] LB 虚拟机 1
                    - [C] LB 虚拟机 2
        - GatewayVM
            - [C] 支持批量编辑子网掩码，网关
            - 边缘网关卡片编辑内容
                - [C] 所属主机不可编辑
                - [C] 切换 TEPIP 池则清空 TEPIP，placeholder 为自动分配
                - [C] 发生实质修改则展示确认弹窗
                - [C] TEPIP vlan 的下拉菜单里不包含自动创建选项
            - 二次修改弹窗内容分区显示
                - [C] 顶部橙色提示只包含管理网络修改
                - [C] 顶部橙色提示只包含 TEP 网络修改
                - [C] 顶部橙色提示包含两种 TEP 网络修改
                - [C] 提示分区：管理网络子网掩码修改，未修改则不显示
                - [C] 提示分区：管理网络网关修改，未修改则不显示
                - [C] 提示分区：边缘网关虚拟机x：管理虚拟机网络，管理 IP 地址，TEP 虚拟机网络，TEPIP，部分边缘网关虚拟机未修改则不显示
                - [C] 自动分配 TEPIP
        - VPC集群关联
            - [C] 编辑集群关联 VDS，需要关联所有主机的网口
            - [C] 编辑 TEPIP 池子后，TEPIP 留空自动填充
            - [C] 头部批量填充，主机名称顺序自上而下排序
            - 二次弹窗
                - [C] 顶部提示框
                - [C] 提示分区：虚拟机分布式交换机
                - [C] 提示分区：TEPIP
                - [C] 自动分配 TEPIP
        - TEPIP池子
            - [C] 子网掩码支持编辑，不可为空
            - [C] 子网掩码编辑后，网关与 TEPIP 地址池需要属于同一网段
