# Everoute VPC 产品知识摘要

## 适用范围

- 本文件用于 Step6-10 的产品知识加载，帮助识别 VPC 相关对象树、详情结构、资源关系和高优先级拆分轴。
- 本文件不直接定义需求范围，只定义如何把 VPC 需求拆成稳定蓝图和最终用例。

## 产品对象树

- 基础对象：TEP IP 池、关联集群、VPC、VPC 子网、路由表、浮动 IP。
- 网关对象：边缘网关、边缘网关组、外部子网、外部子网组、路由网关、NAT 网关、L2 网关。
- 互联对象：VPC 对等连接、下一跳、关联路由表、引用对象。
- 安全对象：VPC 安全组、VPC 安全策略、VPC 隔离虚拟机。
- 高可用对象：主边缘网关、备边缘网关、活动网关、主外部子网、备外部子网。
- 固定块对象：事件审计、权限控制、删除限制、关联限制、其他相关改动。

## 页面与动作树

- 服务设置链路：设置 TEP IP 池 -> 关联集群 -> 编辑或解除关联 -> 边缘网关/组与外部子网/组管理。
- VPC 对象链路：创建/查看/编辑/删除 VPC、子网、路由表、浮动 IP、对等连接。
- 网关服务链路：创建/编辑/删除路由网关、NAT 网关、L2 网关、VPC 对等连接。
- 安全服务链路：创建/编辑/删除 VPC 安全组、安全策略、隔离虚拟机。
- 详情链路：概览、关联资源详情、tab、引用对象、跳转链接、超长名称与说明文案。

## 历史样本子标题习惯

- 顶层经常按业务对象拆：`VPC 对等连接 / VPC 多网关 / 外部子网 / 外部子网组 / 浮动 IP / 边缘网关`。
- 第二层多按页面或动作拆：`列表 / 创建 / 编辑 / 删除 / 详情 / 事件审计 / 权限控制 / 其他相关改动`。
- 在详情页样本中，通常先拆 `tab / 详情按钮 / 基本信息 / 用途分配地址块 / 关联路由表`。
- 在多网关场景中，样本会按 `对象类型 -> 共用/非共用 -> 主备 -> 多 CIDR -> 校验或结果` 继续细分。
- 在资源选择类场景中，样本经常按 `所属 VPC -> 子网/外部子网 -> 过滤范围 -> 字段显隐/禁用 -> 报错/成功` 拆分。

## 高频固定块

- 事件审计与任务：创建、编辑、删除、切换活动网关、关联资源、编辑网络配置。
- 权限控制：只读/可编辑、是否展示操作按钮、是否允许创建或删除。
- 删除限制：被路由规则引用、被网关服务引用、被浮动 IP 引用、被对等连接引用。
- 关联限制：资源属于当前 VPC 或不属于当前 VPC、过滤范围、下一跳可选性、引用状态。
- 文案与其他相关改动：长名称、超长文案、跳转、提示、卡片统计变化。

## 高优先级拆分轴

- 页面动作：列表、创建、编辑、删除、详情、事件审计、权限控制、其他相关改动。
- 对象关系：VPC 与子网、路由表与下一跳、边缘网关与外部子网、边缘网关组与外部子网组、对等连接与关联路由表。
- 共享类型：共享/非共享、共用/非共用。
- 高可用关系：单对象与组对象、主备、活动/非活动、主备地址块是否一致。
- 地址块复杂度：单 CIDR、多 CIDR、CIDR 重叠、CIDR 扩缩容、CIDR 与内部子网重叠。
- 过滤与引用：所属 VPC、引用状态、删除限制、关联限制、过滤范围。
- 结果类型：字段显隐、禁用态、说明文案、格式校验、跳转、创建成功、删除限制、文案变化。
- 许可关系：关联集群许可、额度状态、编辑受限、关联失败、报错归属。

## 资源类型与前提

- 环境资源：ER 版本、Tower 版本、关联集群、TEP IP 池、边缘网关配置规格。
- 对象资源：VPC、子网、路由表、边缘网关、边缘网关组、外部子网、外部子网组、浮动 IP、对等连接、安全服务。
- 网络资源：VLAN、VPC 网络、VPC 子网、TEP IP、网关地址、地址块、下一跳。
- 引用资源：被路由表引用、被网关服务引用、被浮动 IP 引用、被对等连接引用。
- 许可资源：关联集群 CPU 插槽许可、额度状态、文案变化、编辑或关联限制。
- 典型前提：先启用 VPC、先设置 TEP IP 池、先关联集群、先创建边缘网关或组、先具备外部子网或组。

## 复合 case 拆分信号

- 单条 case 同时描述多个页面：列表、详情、弹窗或 tab。
- 单条 case 同时描述主、备两个外部子网或多个 CIDR 校验。
- 单条 case 同时描述过滤、禁用、报错和提交结果。
- 单条 case 使用“多个”“批量”“反复切换”“同时”时，实际覆盖多个对象或多个结果。
- 单条 case 同时覆盖共享/非共享、共用/非共用、主备一致/不一致。
- 单条 case 同时覆盖资源引用状态和删除限制结果。

## 禁止合并项

- 禁止把列表页、详情页、弹窗页检查点混写。
- 禁止把 `字段展示正确 / 字段可编辑 / 过滤正确 / 报错正确 / 提交成功` 压成一条 case。
- 禁止把 `共享/非共享` 与 `主备` 与 `多 CIDR` 混成一个抽象“网络配置正确”。
- 禁止把 `资源属于当前 VPC`、`资源已被引用`、`删除限制` 混成一个通用异常结果。
- 禁止在有 tab、section、详情按钮或关联对象时直接从对象层跳到 `[C]`。

## 术语与版本提醒

- VPC 常见高频术语包括：TEP IP 池、边缘网关/组、外部子网/组、路由表、浮动 IP、下一跳、对等连接。
- 边缘网关组和外部子网组都具有主备关系，相关网络配置通常存在主备一致性或可切换性要求。
- 多网关场景中，共用/非共用、多 CIDR、地址块重叠、主备地址块数量一致是高频约束。
- VPC 相关资源经常带有强引用关系与删除限制，不能只测试成功路径。
- 文案调整、跳转、超长名称和详情 tab 往往在 VPC 样本中占比不低，不能只关注表单提交流程。
