# Everoute 负载均衡产品知识摘要

## 适用范围

- 本文件用于 Step6-10 的产品知识加载，帮助识别 LB 相关对象树、页面动作、拆分轴和复合 case 风险。
- 本文件不直接替代需求文档、Figma 或历史 testcase，不定义当前需求范围，只定义“如何理解和拆开 LB 需求”。

## 产品对象树

- 顶层对象：LB 功能开关、LB 实例（LBI）、LB 实例组（LBG）、虚拟服务、服务器池、服务器池组、健康检查监视器。
- 关联对象：LB 虚拟机详情、关联网络、关联网络组、VIP 地址池、LIP、TEP 网络、VPC 子网、VPC 子网组。
- 高可用对象：主实例、备实例、活动实例、主服务器池、备服务器池。
- 固定块对象：事件审计与任务、权限管理、异常限制、跨产品影响项（尤其是 VPC 资源与许可）。

## 页面与动作树

- 功能启用链路：部署 Everoute 服务 -> 启用 LB -> 创建/查看/编辑 LB 实例 -> 创建/查看/编辑 LB 实例组。
- 网络链路：关联网络 / 编辑关联网络 / 解除关联 -> 按与客户端通信、与服务器池通信、同时通信继续拆分。
- 服务链路：创建虚拟服务 -> 基本信息 -> 网络 -> 服务器池 -> 其他参数 -> 查看 / 编辑 / 删除 / 切换活动服务器池。
- 池链路：创建/编辑/删除服务器池、服务器池组、健康检查监视器。
- 许可链路：许可卡片展示 -> 部署/启用/创建/扩缩容时的额度检查 -> 文案、限制、消耗与归还。

## 历史样本子标题习惯

- 顶层通常按对象拆：`LB / LBI / LBG / 虚拟服务 / 服务器池 / 固定块`。
- 第二层通常按页面或动作拆：`部署 Everoute 服务 / 启用 LB / 创建 / 编辑 / 详情 / 关联网络 / 解除关联 / 删除`。
- 第三层通常按子模块拆：`支持 VPC 开关 / 网络设置 / TEP 网络 / VIP / LIP / 主备关系 / 配置方式 / 卡片或 tab`。
- 在网络类需求里，样本通常继续按 `网络类型 -> 通信方向 -> 字段组 -> 校验或结果` 拆分。
- 在许可类需求里，样本通常按 `场景 -> 关联功能 -> 许可子类 -> 环境对象 -> 额度状态 -> 结果` 拆分。

## 高频固定块

- 事件审计与任务：创建、编辑、删除、切换活动实例、关联网络、解除关联、启用与部署相关动作。
- 权限管理：可编辑/只读、是否展示操作按钮、是否允许提交。
- 异常限制：集群连接异常、实例同步异常、主备不一致、资源被引用、不可关闭支持 VPC 等。
- VPC 影响项：LB/VPC 联动时，VPC 资源过滤、VPC 选择范围、VPC 网络与子网关系、VPC 许可不足文案不能丢。

## 高优先级拆分轴

- 对象类型：LBI、LBG、虚拟服务、服务器池、服务器池组、健康检查监视器不能混写。
- 页面动作：部署、启用、创建、编辑、详情、关联、解除关联、删除、切换活动对象必须独立。
- 版本与能力开关：ER 版本、是否支持 VPC、是否允许关闭支持 VPC。
- 网络模式：VLAN、VPC 子网、VPC 子网组、TEP 网络、FullNAT、DNAT。
- 通信方向：与客户端通信、与服务器池通信、同时通信。
- 主备关系：主、备、主备一致、主备不一致、活动切换。
- IP 配置方式：自动分配、手动分配、部分自动部分手动、复用。
- 许可矩阵：许可对象、许可方式、许可类型、额度状态、过期状态、消耗动作、归还动作、结果归属。
- 结果类型：显隐、默认值、过滤、禁用、校验报错、提交成功、限制提示、文案变化。

## 资源类型与前提

- 环境资源：ER 版本、Tower 版本、关联集群、主机数量、LB 虚拟机数量。
- 网络资源：VLAN 网络、VPC、VPC 子网、VPC 子网组、TEP IP 池、VIP、LIP、网络组。
- 对象资源：LB 实例、LB 实例组、虚拟服务、服务器池、服务器池组、健康检查监视器。
- 许可资源：LB vCPU、LB CPU 插槽、试用/订阅/永久许可、已用/可用额度、过期状态。
- 结果资源：创建成功、启用成功、关联成功、不可关闭、文案变化、报错仅包含某产品。
- 典型前提：先部署或启用 LB、先创建实例或实例组、先关联网络或网络组、先具备服务器池或服务器池组、先满足许可额度。

## 复合 case 拆分信号

- 单条 case 同时描述与客户端通信和与服务器池通信。
- 单条 case 同时描述主实例和备实例，或主备一致/不一致两个结果。
- 单条 case 同时描述显隐、字段校验和提交结果。
- 单条 case 使用“反复”“同时”“多个”“批量”“复合点击”等词，且实际覆盖多个动作或多个资源模式。
- 单条 case 同时包含 VPC 选择、子网选择、IP 分配、提交结果等多个独立子过程。
- 单条 case 同时覆盖许可额度判断和功能结果判断。

## 禁止合并项

- 禁止把 `支持 VPC 开/关` 与 `网络设置显隐` 与 `提交结果` 压成一个 case。
- 禁止把 `VLAN` 与 `VPC 子网` 与 `VPC 子网组` 混成一个网络分组。
- 禁止把 `与客户端通信` 与 `与服务器池通信` 合并为“网络配置正确”。
- 禁止把 `主备一致/不一致`、`活动切换`、`异常状态` 合并为“高可用正常”。
- 禁止把 `自动分配/手动分配/部分自动部分手动` 合并为“IP 分配正确”。
- 禁止把 `许可对象 + 许可方式 + 额度状态 + 结果归属` 缩写成“许可不足报错”。

## 术语与版本提醒

- `LBI` 指 LB 实例，`LBG` 指 LB 实例组，`VIP` 通常面向客户端，`LIP` 面向服务器池，`TEP` 用于隧道网络。
- `支持 VPC` 开关会影响网络类型、字段显隐、过滤范围和关闭限制，不能只当成一个布尔字段。
- LB 实例组由主、备 LB 实例组成；虚拟服务可以属于 LB 实例，也可以属于 LB 实例组；服务器池主备与 LB 主备是两套关系。
- VPC 场景下，网络类型、所属 VPC、子网过滤、通信方向、主备一致性通常会联动变化。
- 许可类需求中，`按 vCPU` 与 `按 CPU 插槽` 是完全不同的展开轴，不能与许可类型混成一个维度。
