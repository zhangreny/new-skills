# Step9 规则门禁报告模板

输出文件 `step9_rule_gate_report.md` 必须包含以下二级标题，且顺序固定：

```md
# Step9 规则门禁报告
## 已加载资料
## 采用的规则
## 继承的蓝图与组合
## 剪枝与未落地组合
## 违规项与修正结果
## 黄金参考对比或跳过说明
```

## 写法要求

- `已加载资料`：列出需求文档、图片、Step6/7/8 产物和引用到的历史样本。
- `采用的规则`：列出 common/product/dimension/resource/golden 规则。
- `继承的蓝图与组合`：说明 Step9 如何承接 Step7 与 Step8。
- `剪枝与未落地组合`：Step8 中任何未落成 `[C]` 的核心组合都必须在这里解释原因。
- `违规项与修正结果`：记录本轮自检发现的问题和对应修正。
- `黄金参考对比或跳过说明`：若采用黄金参考，则写对象覆盖、subsection 覆盖、关键维度轴覆盖和相对数量级；若未采用，则明确写跳过原因。
